# 🫙 Base Tip Jar

Accept crypto tips with minimal fees on Base blockchain.

## 🚀 Features
- Connect Web3 wallet (MetaMask, Coinbase Wallet)
- Send tips with <$0.01 fees
- Instant withdrawals
- Mobile responsive

## 🛠️ Tech Stack
- React / Next.js
- Ethers.js / Viem
- Base (Ethereum L2)
- Vercel

## 📦 Installation
```bash
npm install
npm run dev
```
