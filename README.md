# 🫙 Base Tip Jar

Accept crypto tips with minimal fees on Base blockchain. Create your tip jar, set goals, and receive support from your community.

![Base Tip Jar](https://img.shields.io/badge/Built%20on-Base-0052FF?style=for-the-badge)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

## 🌟 Features

### For Tip Jar Creators
- ✨ **Shareable Tip Pages** - Get your unique URL (`base-tip-jar.app/username`)
- 🎯 **Goals & Milestones** - Set funding goals with visual progress bars
- 💬 **Custom Messages** - Receive encouraging messages with every tip
- 📊 **Analytics Dashboard** - Track tips, supporters, and engagement
- 🎨 **Profile Customization** - Avatar, bio, category, and social links
- 🏆 **Supporter Leaderboard** - Highlight your top supporters
- 📱 **QR Codes** - Easy mobile tipping
- 📤 **Social Sharing** - Share to Twitter, Farcaster, and more

### For Supporters
- ⚡ **Ultra-Low Fees** - Transactions under $0.01 on Base L2
- 💰 **Quick Tip Amounts** - Preset options (Coffee, Lunch, Dinner, Generous)
- 🔍 **Browse Creators** - Discover tip jars by category
- 💙 **Leave Messages** - Send encouragement with your tips
- 🔒 **Non-Custodial** - Tips go directly to creator's wallet

## 🛠️ Tech Stack

- **Framework:** Next.js 13+ (React)
- **Blockchain:** Base (Ethereum L2)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Web3:** Ethers.js / Viem
- **Wallets:** MetaMask, Coinbase Wallet

## 📦 Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/base-tip-jar.git

# Navigate to directory
cd base-tip-jar

# Install dependencies
npm install

# Create .env file
cp .env.example .env.local

# Add your environment variables
NEXT_PUBLIC_CONTRACT_ADDRESS=your_contract_address
NEXT_PUBLIC_CHAIN_ID=8453
NEXT_PUBLIC_RPC_URL=https://mainnet.base.org

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## 🎯 Usage

### Create Your Tip Jar

1. Visit `/create`
2. Fill in your profile information
3. Set your wallet address
4. Add a goal (optional)
5. Share your unique link!

### Receive Tips

1. Share your tip jar URL
2. Supporters connect their wallet
3. They send tips directly to your address
4. Track everything in your dashboard

## 🌐 Pages

- `/` - Homepage
- `/create` - Create your tip jar
- `/browse` - Browse all creators
- `/[username]` - Individual tip jar page
- `/dashboard` - Personal analytics
- `/settings` - Edit your profile
- `/about` - About the project
- `/faq` - Frequently asked questions
